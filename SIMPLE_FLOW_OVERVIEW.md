### Match Score Categories

```
┌─────────────────────────────────────────────────────────┐
│               MATCH SCORE INTERPRETATION                 │
└─────────────────────────────────────────────────────────┘

🟢 90-100% Excellent Match
   → Strong recommendation
   → High compatibility
   → Priority candidate

🟢 80-89% Very Good Match
   → Good recommendation
   → Suitable candidate
   → Consider for interview

🟡 70-79% Good Match
   → Moderate recommendation
   → Review carefully
   → Potential fit

🟡 60-69% Moderate Match
   → Low recommendation
   → Some skills match
   → Consider if few applicants

🟠 50-59% Fair Match
   → Not recommended
   → Major gaps exist
   → Review only if necessary

🔴 0-49% Poor Match
   → Not suitable
   → Significant mismatch
   → Do not recommend
```

### Common Questions

